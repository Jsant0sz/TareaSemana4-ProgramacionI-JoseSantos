**Tarea III - Entradas y Conversioness**

El siguiente programa reailiza una solicitud de datos, los cualos utilizan las varibles
int, double y string. teniendo como caracteristica, para la varible string el usar *getline*
para que se pueda detectar y usar los espacios entre el nombre solicitado, mientras para las
variables double, se uso *fixed y setprecision(2)*, por el motivo de que los datos numericos
se usan para poder calcular gastos economicos, lo cual ayuda a aproximar los decimales que
se necesitan.
